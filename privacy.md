# PocketAI Studio 隐私政策

**最后更新：2025年11月**

## 中文 (Chinese)

### 1. 数据收集
我们不会收集、存储或向我们的服务器传输任何个人数据。所有数据均保留在您的设备上。

### 2. API 密钥存储
- 您的 API 密钥（OpenAI、DeepSeek、Qwen、Gemini）使用 iOS 安全存储（UserDefaults）本地保存在您的设备上。
- API 密钥永远不会传输到我们的服务器。
- API 密钥仅直接发送到您配置的相应 AI 服务提供商。

### 3. 项目数据
- 所有项目和代码均本地存储在您的设备上。
- 运行日志本地存储用于调试目的。
- 我们无法访问您的项目内容。

### 4. 第三方 AI 服务
当您使用 AI 功能时，您的提示词和代码会直接发送到您配置的 AI 提供商。请查阅他们的隐私政策。

### 5. 数据安全
- 所有数据使用 iOS 安全存储机制本地存储。
- 与 AI 提供商的网络通信使用 HTTPS 加密。
- 您可以通过卸载应用来删除所有数据。

### 6. 您的权利
- 您对自己的数据拥有完全控制权。
- 您可以随时删除项目和日志。
- 您可以从设置中删除 API 密钥。

### 7. 儿童隐私
本应用不会有意收集 13 岁以下儿童的数据。

### 8. 政策变更
我们可能会更新此政策。继续使用即表示接受更改。

### 9. 联系方式
如有疑问：这是一个开源项目。请参阅代码仓库获取联系信息。

## English

### 1. Data Collection
We do not collect, store, or transmit any personal data to our servers. All data remains on your device.

### 2. API Key Storage
- Your API keys (OpenAI, DeepSeek, Qwen, Gemini) are stored locally on your device using iOS secure storage (UserDefaults).
- API keys are never transmitted to our servers.
- API keys are only sent directly to the respective AI service providers you configure.

### 3. Project Data
- All projects and code are stored locally on your device.
- Run logs are stored locally for debugging purposes.
- We cannot access your project content.

### 4. Third-Party AI Services
When you use AI features, your prompts and code are sent directly to your configured AI providers. Please refer to their privacy policies.

### 5. Data Security
- All data is stored locally using iOS secure storage mechanisms.
- Network communication with AI providers uses HTTPS encryption.
- You can delete all data by uninstalling the app.

### 6. Your Rights
- You have full control over your data.
- You can delete projects and logs at any time.
- You can remove API keys from settings.

### 7. Children's Privacy
This app does not knowingly collect data from children under 13.

### 8. Policy Changes
We may update this policy. Continued use signifies acceptance of changes.

### 9. Contact Information
For questions: This is an open-source project. Please refer to the code repository for contact information.
